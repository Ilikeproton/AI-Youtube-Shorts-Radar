# AI YouTube Shorts Radar

This `app.zip` release is the Windows desktop build of **AI YouTube Shorts Radar**.

## What It Does

AI YouTube Shorts Radar helps Shorts creators find the next breakout topic faster.

- Start with one seed keyword
- Expand more directions with free AI
- Find rising and high-view Shorts
- Save research locally on your own machine

## System Requirements

- Windows 10 or Windows 11
- WebView2 Runtime installed
- Internet connection
- A free YouTube Data API key

## How To Run

1. Extract `app.zip`
2. Open `app.exe`
3. In `App Settings`, add your YouTube API key
4. Optionally add a proxy
5. Enter a seed keyword
6. Click `Create and Run`

## Proxy Field

You can leave the proxy field empty.

You can also enter values like:

- `127.0.0.1:10601`
- `socks5://127.0.0.1:10601`
- `http://127.0.0.1:10601`

If you enter `127.0.0.1:10601`, the app will try SOCKS5 first and fall back to HTTP.

## Local Data

This release is local-first.

- The app creates its database automatically in the same folder as `app.exe`
- Your research stays on your machine
